# 8051 Microcontroller Practice Projects

## 專案簡介 (About)
這是微處理機實習課程的系列專案。主要使用 8051 系列微控制器（如 AT89S51）進行硬體控制與韌體開發。專案涵蓋了基礎的 GPIO 控制、計時器中斷（Timer Interrupts）應用以及周邊硬體驅動，展示了對底層硬體架構的理解以及使用 C 語言與組合語言的實作能力。

## 硬體與技術棧 (Tech Stack)
* **微控制器 (MCU):** AT89S51 / 89C51
* **開發語言:** C 語言、組合語言 (Assembly)
* **周邊硬體:** LED 陣列、8Ω 0.5W 揚聲器、七段顯示器、按鍵開關

---

## 專案列表 (Projects Overview)

### 📁 1. LED 燈光控制 (LED Pattern Control)
* **說明:** 透過控制 I/O Port，實作不同的 LED 亮滅模式與跑馬燈效果。
* **技術重點:**
    * 透過 P0, P1, P2, P3 輸出不同燈碼。
    * 使用暫存器與巢狀迴圈，實作出 0.1秒、0.25秒與0.5秒的精確延遲子程式 (`DELAY_01S`, `DELAY_025S`)。
    * 原始碼使用組合語言撰寫。

### 📁 2. 七段顯示器控制 (7-Segment Display)
* **說明:** 控制七段顯示器以動態掃描的方式顯示特定的數字（學號 0106C033）。
* **技術重點:**
    * 使用 C 語言進行開發 (`<reg51.h>`)。
    * 利用陣列儲存七段顯示器的 16 進位顯示碼 (例如 `0x3F`, `0x06` 等)。
    * 透過軟體延遲迴圈 (Delay loop) 達到視覺暫留的顯示效果。
 
### 📁 3. 聲音產生器 (Music Player)
* **說明:** 使用 8051 單片機實現音樂播放功能，驗證定時器與揚聲器硬體的正常運作。
* **技術重點:**
    * 利用 Timer 0 與 Timer 1 產生中斷，精確控制音調頻率與節拍持續時間。
    * 查表法 (Lookup Table)：將音調數據寫入記憶體 (`TABLE`)，根據查表轉換對應的頻率輸出至揚聲器。
* **原始碼:** 使用組合語言撰寫。
